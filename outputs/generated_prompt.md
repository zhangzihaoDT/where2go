## 判断标准

请基于实时搜索结果，筛选 TOP5 适合假期出行的目的地。

判断标准：

### 1. 酒店涨幅

- 端午价 / 平日价 ≤ 1.25：可冲
- 酒店涨幅 25%–50%：只适合强目的地
- 酒店涨幅 >50%：除非特别想去，否则放弃

### 2. 机票 / 交通价格

- 机票往返涨幅 ≤20%：可接受
- 如果高铁 / 自驾更合适，可以不使用机票作为主要判断
- 但必须说明主要交通方式和预计交通耗时

### 3. 交通时间

- 推荐目的地必须考虑离出发地的距离
- 2 天行程不推荐单程交通时间过长的目的地
- 3 天行程可以接受中等交通距离
- 4 天以上才适合明显长线目的地

### 4. 游玩时间

- 必须考虑当地核心景点需要多少时间消化
- 如果目的地本身需要 4–5 天才能玩好，不要推荐给 2–3 天短假
- 如果景点太分散，也要降低推荐优先级
